¿Qué es el Docker?

Es un software que realiza una recolexion, a nivel del sistema operativo.
Nos permite crear sistemas operativos, dentro de otros que estamos usando (linux, ubuntu, etc.).

Ventajas

-Docker aisla el sistema operativo interno, a nivel del Kernel, permitiendo el uso de estos sistemas operativos en nuestras maquinas.
-Se comparte con el Kernel y las librerias, todo lo requerido ya esta presente para el funcionamiento del Docker, consume menos recursos del sistema que otros programas, permitiendo crear gran numero de maquinas virtuales, sin causar problemas de consumo.

Desventajas

-Hay incompatibilidad con otros sistemas que no tengan el Kernel, para su funcionamiento en otros sistemas sera nesesario tener Kernel y arreglar la conexion al Docker
-Los sistemas operativos virtuales que se crean son Linux.

Recursos

Utiliza recursos a nivel de kernel y dos herramientas de aislamiento general(cgroup y namespaces)
Quiere decir que los ficheros de la maquina virtual pueden ser accedidos desde el host físico, pero la maquina virtual no puede acceder a nada fuera de ella.
Permiten aislar el cpu y memoria que se va a usar, personalizandolo hasta ciertos limites

Utilidad

Docker consige ser flexible y vastante portable, permitiendo migrar estos sistemas operativos de una maquina a otra, copiar, extender etc.
Se puede crear un registro en cloud de un Docker que permitira usarlo desde cualquier lugar con acceso al internet.
Esta enfocado a sistemas distribuidos, creando copias de entorno TI
