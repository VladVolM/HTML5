Instalacion para Ubuntu

Tener en cuenta algunos prerequisitos de tu sistema:
-Tener una instalación de 64 bits.
-Usar la distribución actualizada del Kernel

Borrar version antigua de Docker

En tu consola de comandos puedes usar "uname -r" para ver la versión(ej. 4.4.0-36-generic)

$ sudo apt-get remove docker docker-engine docker.io containerd runc

Crear el repositorio

Hace falta hacer un update al "apt sources"

$ sudo apt-get update


Instalar paquetes que permiten usar repositorios a traves de https:

$ sudo apt-get install \
apt-transport-https \
ca-certificates \
curl \
gnupg-agent \
software-properties-common


Añadir la lave oficial de Docker:

$ curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -
#Para comprobar la llave, que tiene que ser la siguiente se usa: (9DC8 5822 9FC7 DD38 854A E2D8 8D81 803C 0EBF CD88)
$ sudo apt-key fingerprint 0EBFCD88
#que daria el siguiente resultado:
pub rsa4096 2017-02-22 [SCEA]
9DC8 5822 9FC7 DD38 854A E2D8 8D81 803C 0EBF CD88
uid [ unknown] Docker Release (CE deb) <docker@docker.com>
sub rsa4096 2017-02-22 [S]


Para preparar un estable repositorio se usaria:

$ sudo add-apt-repository \
"deb [arch=amd64] https://download.docker.com/linux/ubuntu \
$(lsb_release -cs) \
stable"


Donde el codigo cambia segun sistemas y distribudor(usar la pagina web de descarga para mas detalles)

Instalación

Tras haber preparado el repositorio se vuelve a usar

$ sudo apt-get update


Instalar la ultima vercionde docker:

$ sudo apt-get install docker-ce docker-ce-cli containerd.io


Pero si quieres instalar una vercion espesifica se usaria

$ apt-cache madison docker-ce
Que daria algo como esto
docker-ce | 5:18.09.1~3-0~ubuntu-xenial | https://download.docker.com/linux/ubuntu xenial/stable amd64 Packages
docker-ce | 5:18.09.0~3-0~ubuntu-xenial | https://download.docker.com/linux/ubuntu xenial/stable amd64 Packages
etc.


Teniendo en cuenta el apartado de 5:18.09.1~3-0~ubuntu-xenial, se usaria el espesifico en el siguiene comando;

$ sudo apt-get install docker-ce=<STRING> docker-ce-cli=<STRING> containerd.io


Donde sustituyes STRING por la vercion.
al final usa

$ sudo docker run hello-world


si no da errores esta preparado, (en la pagina web te indican apartados adicionales para descargar paquetes, y hacer update al Docker para futuro uso propio, fuera de esta clase)

$ service docker status $ systemctl status docker


se pueden usar para estar mas seguro al ver su estado

$ docker version
